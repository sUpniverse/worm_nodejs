# 패키지 매니저

- npm 시작과 패키지 만들기(package.json)
  - npm init
  - 만들어진 .json 파일에는 설명들이 있다.
- 패키지 설치하기
  - npm install <패키지명>
  - --save--dev, -D
    - 배포환경이 아닌 개발환경에서만 패키지를 쓰고싶을때
  - global, -g
    - 명령어로 쓸수있는 패키지를 설치하고 싶을때
- Semantic Version, SemVer
  - major.minor.patch

- 명령어
  - npm update
  - npm outdate
  - npm remove
  - npm search
  - npm info <패키지명>
  - npm is <패키지명>
- 패키지 배포하기
  - npm adduser
  - npm bullish
  - npm unpublished <패키지명>
    - 24시간 이내에 가능
